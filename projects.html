Developed four different applications for the User Interfaces class at Waterloo (CS 349), all of which employ the Model-View-Controller paradigm. Note that all were developed specifically for use in Google Chrome



Utrack

HTML CSS JAVASCRIPT

Utrack is an application that allows users to track their daily activies and various health measures, and analyze their data through data summaries including tables, lists and bar graphs.

Fotag

HTML CSS JAVASCRIPT

Fotag is an image rating web application that allows users to display a set of images using different views. Users can rate images and sort based on given ratings. (Currently only works locally with images from the image directory)

Buggui

HTML CSS JAVASCRIPT

Buggui is an application designed using the HTML5 canvas in which a car can be directly manipulated using the mouse through affine transforms. The user can:

  • Position it on the screen by selecting it and dragging it with a pointer
  • Rotate it about its center by dragging the mouse pointer
  • Make it wider or longer by interactively manipulating control points on the car
  • Make the axles longer by interactively pulling the tires out
  • Turn the car's front wheels by dragging them with the mouse

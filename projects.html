Developed four different applications for the User Interfaces class at Waterloo (CS 349), all of which employ the Model-View-Controller paradigm. Note that all were developed specifically for use in Google Chrome



Utrack

Utrack is an application that allows users to track their daily activies and various health measures, and analyze their data through data summaries including tables, lists and bar graphs.

Fotag

Fotag is an image rating web application that allows users to display a set of images using different views. Users can rate images and sort based on given ratings. (Currently only works with images from ./images)
